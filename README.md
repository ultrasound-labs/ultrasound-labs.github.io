# Ultrasound Labs Website

Welcome to the Ultrasound Labs website! This project is designed to showcase our vision and services in a professional manner, utilizing a theme inspired by bats and ultrasound waves.

## Project Overview

The Ultrasound Labs website is a static, single-page application hosted on GitHub Pages. It features a clean and modern design, ensuring an engaging user experience. The website is structured to provide easy access to our core offerings and mission.

## File Structure

- **index.html**: The main HTML file that contains the layout and content of the website.
- **404.html**: A custom error page for handling non-existent routes.
- **css/styles.css**: Stylesheet for the website, defining the visual aesthetics.
- **js/main.js**: JavaScript file for interactive features and enhancements.
- **data/content.json**: JSON file for structured content management, including vision and services.
- **CNAME**: Configuration file for custom domain settings on GitHub Pages.
- **.nojekyll**: File to prevent Jekyll processing on GitHub Pages.
- **LICENSE**: Licensing information for the project.

## Getting Started

To get started with the Ultrasound Labs website:

1. Clone the repository to your local machine.
2. Open `index.html` in your web browser to view the website.
3. Modify the content in `data/content.json` to update the vision and services as needed.
4. Customize styles in `css/styles.css` to align with your branding.
5. Deploy to GitHub Pages by pushing your changes to the `main` branch.
